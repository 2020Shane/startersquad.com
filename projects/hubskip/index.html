Hubskip

Hubskip helps you find the destinations that match your interest while getting you the best price for your flight.

Project description

Hubskip sells city trips to all major destinations in world. They focus on a prediction system for flight prices. StarterSquad developed the online shop, including search, payment and integration with several b2b providers.

Within 8 weeks we have built the full solution, right on time to demo at the Launch Festival 2013

Front-end

The front-end is based on our AngularJS + RequireJS seed project. The combination of RequireJS and Angular allows us to build large code bases without worrying about ordering in the script tags. In addition it allows for compression of the javascript into a single or several minified files.

AngularJS helped us to keep the application code _really_ simple. Some metrics: max number of lines in a single controller is 257, average number is 91 lines and the median is less then 50 lines! You can say: "Yea, bla-bla... These guys moved most of their logic to services then". Nope. Maximum service file size is 129 lines and average is 51. Can you say KISS?

Back-end

The REST backend was done in Scala with Spray.

It was a hard choice among various technologies, but in the end the deciding factor was the fact that both Scala and Spray begin with the letter S. As usual we were right - they made a perfect fit.
